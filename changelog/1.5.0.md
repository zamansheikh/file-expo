## Forgot your password? Reset it from the VPS

A break-glass recovery script for when you can't log into the web panel — run it over
SSH and set a new password directly on the server.

- **New: `reset-password.sh`** — rewrites the panel password in `/etc/file-expo/config.json`
  using the exact same hashing the server uses, then restarts the service
- **No clone needed** — run it straight from GitHub, just like the installer:

  ```
  curl -fsSL https://raw.githubusercontent.com/zamansheikh/file-expo/main/reset-password.sh | sudo bash
  ```

  Or, if you already have the repo on the box:

  ```
  sudo bash reset-password.sh                 # prompts for a new password (hidden + confirmed)
  sudo bash reset-password.sh 'newpassword'   # non-interactive
  ```

- **Safe by design** — requires root, refuses passwords under 6 characters, invalidates any
  pending setup link, and restarts the service so old sessions are dropped
- Clear errors if Node isn't installed or File Expo isn't set up on that machine
