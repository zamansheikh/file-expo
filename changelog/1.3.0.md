## PM2 manager & full domain config control

- **New: PM2 tab** (Tools → PM2)
  - Lists all PM2 processes with status, CPU, memory, restarts and uptime
  - Start / stop / restart / delete, view logs
  - **Start a new app** right from the panel — a script path (`server.js`) or any command (`npm run start`) with a name and working folder
  - **Save list** button (`pm2 save`) so apps come back after a reboot
  - One-click **Install PM2** if it's not on the server yet
- **Domains: detects ALL nginx sites now** — not just ones created by File Expo. Manually-created configs show up with a "detected" badge
- **Edit any domain config** — ✎ Edit opens the nginx config file in the built-in editor
- **Enable / Disable sites** (sites-enabled layout) with automatic config-test and rollback if the config is broken
- **Test & reload nginx** button — validates the config (`nginx -t`) and reloads safely; a broken config is never applied
