## Real terminal — replaces the old console

The old console ran every command in a **separate** shell, so `cd` never stuck, interactive
programs failed, and output only appeared once a command finished. It's now a **real terminal**.

- **True PTY** — a persistent shell over a WebSocket. `vim`, `nano`, `top`, `htop`, `less`,
  `sudo` password prompts, `npm`, `docker`, `git` — all work like a native SSH session
- **Live streaming output** — characters arrive as they're printed, no more waiting for a
  command to finish (long builds, `tail -f`, install logs all stream in real time)
- **State persists** — `cd`, environment variables, shell history and aliases all stick
- **Full colors** (16 / 256 / true-color), cursor control and alternate-screen support
- **Resizable** — drag the top edge, or hit ⛶ to maximize. The shell is told the new size,
  so full-screen apps redraw correctly
- **Type directly into the terminal** — real keyboard input including Ctrl+C, Ctrl+D, Ctrl+L,
  arrows, Tab completion, Home/End, PageUp/Down, F-keys, Alt combos
- **Scrollback** — mouse-wheel up through 1500 lines of history
- **cd here** button jumps the shell to the folder open in the explorer
- Copy with Ctrl+C on a selection, paste with Ctrl+V

Zero new dependencies: the WebSocket layer, PTY bridge and terminal emulator are all built in.
