## First release 🎉

- One-line installer (`curl | sudo bash`) with OS-style terminal loader
- Browser setup wizard — create your admin password via the setup link
- Full GUI file manager: browse, upload (drag & drop), download, edit, rename, copy/cut/paste
- Permissions editor, properties, compress/extract, deep search
- Built-in console panel
- Login protection with scrypt-hashed passwords and rate limiting
- Runs as a systemd service, auto-starts on reboot
- Supports Ubuntu/Debian, Arch, Fedora/RHEL, Alpine, openSUSE
