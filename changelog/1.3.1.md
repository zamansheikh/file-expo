## tmux session manager

- **New: tmux tab** (Tools → tmux)
  - Lists all tmux sessions (windows count, attached/detached)
  - **Live screen view** — click a session to see its pane output, auto-refreshing every 2 seconds
  - **Send commands** into a session by typing and pressing Enter — great for background jobs
  - Quick keys: send **Ctrl+C** and **↑** (last command) to the session
  - **Create** a new session (optionally running a command, in the current folder) and **kill** sessions
  - One-click **Install tmux** if it's not on the server
- Keep long-running tasks alive in the background, then check on them any time from the browser
