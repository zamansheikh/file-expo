## Views, self-update, server tools & domains

- **Views** — installable preview add-ons: image, video (with seeking), audio, PDF, Markdown, CSV table, JSON, hex viewer
- **One-click updates** — an Update button appears in the explorer when a new version is pushed to GitHub
- **Server Tools**:
  - Overview dashboard: CPU load, RAM, disk usage, uptime
  - Services: start/stop/restart systemd services, view logs
  - Processes: top by CPU, kill button
  - Ports: all listening ports
  - Cron: edit crontab in the browser
- **Domains** — attach a domain (auto-installs nginx), reverse proxy or static site, one-click Let's Encrypt HTTPS
