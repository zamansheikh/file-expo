## Trash, share links, command palette & more

- **Trash with Undo** — deletes go to Trash with an instant Undo toast; restore or purge from the Trash panel. `Shift+Del` = permanent
- **Share links** — public download URLs with optional expiry, no login needed; manage in Tools → Shares
- **Command palette** — `Ctrl+K` to run any action, jump to places/bookmarks/files, or type a `/path`
- **Grid view** with real image thumbnails
- **Content search** — grep inside files with matched-line snippets
- **Docker tab** — containers, start/stop/restart/remove, logs
- **Git aware** — branch chip in the toolbar, status/log, Pull & Fetch
- **Folder size analyzer** — du-based bar breakdown
- Multi-select download as one archive, duplicate file, copy path, console history, transfer speed
